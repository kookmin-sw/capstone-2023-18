Kart Skins - Karting Microgame Add-Ons
========================================

This Microgame Add-On requires:
- "Karting Microgame" Version 4.0.0 available from the Unity Hub.
- In the Unity Hub, select the "Projects" tab, click the "New Project" button, from the list of Templates, select the "Karting Microgame".

You can find more information about this Microgame Add-On here: http://u3d.as/1Xjj

This Microgame Add-On also requires at least one of the following Karts:
- Roadster Kart - Karting Microgame Add-Ons: http://u3d.as/1Mtc
- 4x4 Kart - Karting Microgame Add-Ons: http://u3d.as/1MAA
- Muscle Kart - Karting Microgame Add-Ons: http://u3d.as/1MAB
